2017 新年快乐，少年们

发表于 2016-12-31 | 分类于 杂鱼 |

HI ～

同学们，同事们，基佬们，新年快乐！

这是我的第一篇博文。是的你没看错，我写了一个 blog 来祝你新年快乐。

我是不是闲的蛋疼诚意满满。

这个 blog 的名字叫做 Trash，很符合以后我要在里面写的东西。

连 blog 的图腾都是我从 Ubuntu 里抠出来的。

喏，好大一个 Trash (垃圾桶)。也许等过年的时候我应该把它换成红色的。

以后这里会作为一个技术向的博客而存在，我有空会写一些外行看不懂，内行又觉得漏洞百出的技术文章来和大家装逼探讨。

有人问建立这个博客的成本。

其实我就花了十几块钱买了一个域名，然后把它指向了我的 Github Pages。

Github Pages 虽然免费但是慢到怀疑人生，我又用了 Coding.net 的 Coding Pages 并且把非海外的请求指向它，加载速度才稍稍好了一点。

Coding.net 真的良心，我以后再也不叫你抠腚网了。

如果你也需要搭一个 blog，我可以提供全套技术咨询，除了一点：不许用我现在用的模板。

现在想想，双十一没有买阿里云打五折的云服务器，真是万幸。

如果买了，现在只能去吃土了。

如果哪天我真的没钱吃饭了，我也许会添加一个“打赏”功能。

看到这里的一定都是真爱，最后祝你提前身体健康。
